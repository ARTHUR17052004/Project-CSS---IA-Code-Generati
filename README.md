# Project-CSS---IA-Code-Generati
A CSS project aimed at developing an AI code generator for programming languages.

Nome do projeto--->

<><CSS Generator AI><>

>Descrição>

Aplicação web desenvolvida para gerar trechos de código CSS a partir de descrições em linguagem natural. O projeto simula a experiência de um gerador de código com IA, servindo como base para futura integração com APIs de inteligência artificial.

>Tecnologias>

HTML5
CSS3
JavaScript

>Funcionalidades>

campo de descrição textual
botão para gerar código
exibição dinâmica do CSS gerado
interface moderna com tema dark neon

>Próximos passos>

integração com API de IA
cópia rápida do código
suporte a HTML/CSS/JS
histórico de prompts
